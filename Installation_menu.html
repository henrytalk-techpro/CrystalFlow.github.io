Crystal Flow AI 媒體分析器使用手冊

1. 概述

Crystal Flow AI 媒體分析器是一款智能多媒體數據分析工具。它支援處理各種格式的數位內容，包括 YouTube 影片、音訊檔案、文件（TXT、PDF、PPT、EXL、Word 等），並透過心智圖、文字稿和雙語翻譯幫助用戶高效分析和管理資訊。

2. 系統需求

  • 作業系統：Windows 10 或更高版本
  • 硬體需求：
    • 最低配置：雙核心 CPU、4GB RAM
    • 建議配置：四核心 CPU、8GB RAM、NVIDIA GPU（支援 CUDA，選配）
  • 必要軟體：
    • Node.js 安裝指南
    • Markdown 檔案閱讀器：VSCode 或瀏覽器外掛（Chrome、Microsoft Edge）
    • ffmpeg：開源外掛，程式會自動下載
    • OpenAI API Key 指南
    • OpenRouter API Key 指南
    • LM Studio 安裝指南
    • Ollama 安裝指南

3. 功能特色

a. 雙重使用者介面支援

  • Windows 使用者介面
    • 提供完整的多媒體分析功能，專為處理 PC 上的多媒體資訊設計。
    • 支援處理音訊、影片、文件和心智圖生成。
  • 網頁介面
    • 支援透過 QR Code 掃描從行動裝置快速上傳資料至 PC。
    • 使用者可直接在行動裝置上查看進度，並與 PC 工具無縫協作。

b. 線上多媒體知識分析

  • 支援多種來源和格式
    • YouTube：自動處理公開或私人影片內容，提取字幕進行分析。
      • 使用案例：學術課程筆記、技術示範摘要。
    • Facebook 影片：處理分享連結，將影片內容轉換為視覺化資料。
      • 使用案例：會議記錄、品牌廣告分析。
    • 任何 URL：支援解析數位文章（新聞、電子報、維基百科）。
      • 使用案例：研究核心內容提取。
  • 影片字幕下載和翻譯
    • 自動從 YouTube 和 Facebook 提取字幕。
    • 提供多語言翻譯，匯出為 TXT 和 Markdown（MD）檔案。
    • 可生成 HTML 格式心智圖供瀏覽器檢視。

c. 文字和圖像處理

  • 文字處理
    • 支援從 PDF、PPT、Word 和 Excel 檔案提取內容，轉換為 Markdown 格式。
      • 使用案例：學術研究整理、商業報告準備。
    • 準確保留表格、段落和標題版面配置。
    • 使用者可直接生成心智圖或進行多語言翻譯。
  • 圖像處理
    • 支援 jpg、jpeg、png、bmp、heic、webp 格式。
    • 從圖像或掃描文件提取文字，轉換為 Markdown 格式。
      • 使用案例：手寫筆記數位化、圖像文字分析。
    • 整合 Google Lens/Microsoft Lens 實現快速掃描和文字轉換。

d. 本地音訊和 MP3 音樂處理

  • 支援 MP3、M4A、ACC 和 WAV 格式。
  • 提取歌詞並提供雙語翻譯。
    • 使用案例：歌曲翻譯學習、音樂資料整理。
  • 使用語音轉文字技術生成 Markdown 格式文字稿。
  • 可從文字稿或歌詞結構創建心智圖。

e. PC 錄音功能

  • 即時音訊錄製和處理，支援自動生成文字稿。
    • 使用案例：會議錄音分析、演講文字稿生成。
  • 可自訂錄音時長、背景錄音和時間戳記標記。

f. 心智圖生成

  • 可自訂深度層級和結構化呈現。
    • 使用案例：學術研究、會議記錄、創意思考。
  • 結合 GPT 分析並支援多語言。

g. 雙語翻譯和文字轉語音

  • 支援多語言翻譯和雙語並列輸出。
    • 使用案例：國際會議資料準備、多語言學習輔助。
  • 提供文字轉語音功能，可調整速度、音調和聲音風格。
  • 匯出為 Markdown、TXT 或 PDF 檔案。

5. 使用指南

初始設定

  • 啟動軟體後輸入有效的 OpenAI API Key。
  • 確保已安裝 NVIDIA 驅動程式（如果啟用 GPU 加速）。
  • 在 Google Sheets 註冊電子郵件以啟用試用期。

核心功能使用

  • 分析 YouTube 或 URL：將影片連結貼入「輸入媒體 URL」欄位。
  • 選擇要下載的字幕語言。
  • 點擊「分析文字」按鈕。

心智圖生成

  • 選擇心智圖深度層級。
  • 點擊「分析文字」，生成的心智圖會自動儲存。

錄音和文字稿生成

  • 點擊錄音按鈕開始錄音。
  • 錄音完成後自動生成文字稿。

6. 進階功能

  • 多語言解析和翻譯支援。
  • 動態分段處理和 token 數量調整。
  • 與 GPT-4o-mini 互動。

8. 常見問題（FAQ）

  1. 按鈕無法點擊？
    • 檢查電子郵件是否已註冊並授權。
  2. 心智圖太簡單？
    • 減少 token 數量或增加深度層級。
  3. 錄音功能無法使用？
    • 檢查麥克風連接和系統設定。
  4. 如何匯出結果？
    • 使用 Markdown 編輯器或擴充功能檢視生成的檔案。
