10.1 基本功能介紹

圖10-1：檔案伺服器主介面

檔案分享中心

  • 家庭檔案集中管理：
    • 所有上傳的檔案集中顯示
    • 支援檔案預覽和下載
    • 自動顯示檔案大小
  • 檔案組織功能：
    • 依時間排序：最新上傳檔案優先顯示
    • 依名稱排序：方便檔案查找
    • 檔案資訊一目了然

10.2 使用場景

家庭文件共享

  • 家庭成員間快速傳檔：
    • 手機拍照直接上傳
    • 文件即時分享
    • 無需額外軟體
  • 跨設備存取：
    • 電腦、手機、平板都能使用
    • 支援所有現代瀏覽器
    • 無需安裝 App

實用情境

  • 家庭文件整理：
    • 收據、發票掃描存檔
    • 重要文件備份
    • 家庭相片整理
  • 學習資料分享：
    • 作業檔案傳輸
    • 學習筆記分享
    • 教材快速傳遞

10.3 網路設定建議

圖10-2：網路設定指南

基本設定

  • 路由器設定：
    • 建議使用 5GHz WiFi
    • 放置路由器在中心位置
    • 避免訊號干擾
  • 連線方式：
    • 優先使用 HTTPS 連線
    • 支援 HTTP 備用連線
    • 透過 QR Code 快速連接

安全建議

  • 網路安全：
    • 僅在家庭網路中使用
    • 設定 WiFi 密碼保護
    • 定期更改網路密碼
  • 檔案安全：
    • 定期清理不需要的檔案
    • 重要文件及時備份
    • 注意隱私資料保護

10.4 效能優化建議

檔案管理

  • 定期整理：
    • 刪除不需要的檔案
    • 整理檔案分類
    • 維持儲存空間充足
  • 使用建議：
    • 大檔案建議分批上傳
    • 避免同時大量下載
    • 注意設備儲存空間

網路優化

  • 傳輸效能：
    • 保持適當的裝置距離
    • 避開網路尖峰時段
    • 監控網路使用狀況
  • 多人使用：
    • 合理分配網路資源
    • 避免同時大量傳輸
    • 設定檔案大小限制

10.5 常見問題解決

圖10-3：常見問題解決指南

連線問題

  • 無法存取伺服器：
    • 檢查網路連接
    • 確認設備在同一網路
    • 檢查防火牆設定
  • 傳輸速度慢：
    • 檢查網路頻寬
    • 確認檔案大小
    • 考慮使用有線連接

檔案問題

  • 檔案無法上傳：
    • 檢查檔案大小限制
    • 確認儲存空間充足
    • 檢查檔案格式支援
  • 檔案無法預覽：
    • 確認檔案格式
    • 檢查瀏覽器支援
    • 嘗試下載後查看
