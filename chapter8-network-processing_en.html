8.1 Process Network Info. Feature Description

Figure 8-1: Network Media Processing Interface Overview

Supported Network Media

YouTube Videos

  • Automatically detects if subtitles are available
  • With subtitles: Directly download and process subtitles
  • Without subtitles: Use Whisper model to transcribe audio

Facebook Content

  • Supports regular videos
  • Supports Facebook Reels
  • Automatically uses Whisper model to process audio

8.2 Processing Workflow

Figure 8-2: Network Media Processing Workflow

Step 1: Enter Media URL

  • Paste the link in the "Enter Media URL" field
  • Supports links from YouTube, Facebook and other platforms
  • Automatically identifies link type

Step 2: Select Processing Options

  • Download Subtitles: Download existing subtitles
  • Transcription Mode: Speech-to-text mode
  • Select target language

Step 3: Start Processing

  • Click "Process Network Info." to start processing
  • Automatically determines processing method
  • Shows processing progress
  • Automatically outputs results when completed

8.3 Usage Considerations

URL Format

  • Ensure complete video URL is entered
  • Supports standard sharing link formats
  • Avoid using shortened URLs

Processing Time

  • Depends on video length
  • Longer when using Whisper model
  • Recommend reserving sufficient processing time

Network Requirements

  • Requires stable network connection
  • Recommend using high-speed internet
  • Avoid network interruptions

8.4 Common Issue Troubleshooting

Figure 8-3: Common Issue Troubleshooting Guide

Invalid URL

  • Check if URL is complete
  • Confirm video is publicly accessible
  • Check regional restrictions

Download Failed

  • Check network connection
  • Confirm sufficient system resources
  • Retry download process

Processing Interrupted

  • Check network stability
  • Confirm system hasn't entered sleep mode
  • Restart processing

8.5 Best Practice Recommendations

Performance Optimization

  • Avoid processing multiple long videos simultaneously
  • Regularly clean temporary files
  • Monitor system resource usage

Result Management

  • Establish appropriate file naming conventions
  • Regularly backup important content
  • Organize output files
