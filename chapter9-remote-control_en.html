9.1 Connection Setup

Figure 9-1: QR Code Connection Interface

Connection Methods

  • Scan QR Code to connect
    • QR Code is displayed when program starts
    • Supports both HTTP and HTTPS connection methods
    • Recommend using the same local network (WiFi)
  • Ports:
    • HTTP: Default port 11435
    • HTTPS: Default port 11798 (HTTP port + 363)

9.4 Usage Recommendations

Network Settings

  • Ensure mobile phone and computer are on the same network
  • Check firewall settings
  • Prioritize HTTPS connection

Performance Optimization

  • For large files, recommend using WiFi upload
  • Avoid uploading too many files simultaneously
  • Regularly clean temporary files

Security Recommendations

  • Only use on trusted networks
  • Close connection after use
  • Regularly update program version

9.5 Common Issue Resolution

Figure 9-3: Common Issue Resolution Guide

Connection Issues

  • Unable to scan QR Code
    • Confirm camera permissions
    • Check QR Code clarity
    • Try manually entering URL
  • Connection failed
    • Confirm network connection
    • Check firewall settings
    • Confirm ports are open

Upload Issues

  • File upload failed
    • Check file size
    • Confirm network stability
    • Retry upload process
  • Slow processing speed
    • Check network speed
    • Confirm system resources
    • Consider batch processing
