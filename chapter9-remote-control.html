9.1 連線設定

圖9-1：QR Code連線介面

連線方式

  • 掃描 QR Code 連接
    • 程式啟動時會顯示 QR Code
    • 支援 HTTP 和 HTTPS 兩種連線方式
    • 建議使用同一個區域網路（WiFi）
  • 連接埠：
    • HTTP：預設 11435 埠
    • HTTPS：預設 11798 埠（HTTP 埠 + 363）

9.2 手機端功能

媒體上傳

  • 即時錄音
    • 直接用手機錄音
    • 自動上傳至電腦處理
    • 支援長時間錄音
  • 拍照上傳
    • 拍攝文件或圖片
    • 自動進行文字辨識
    • 支援多頁拍攝
  • 文件掃描
    • 類似掃描器功能
    • 自動調整角度和對比度
    • 適合文件數位化

遠端控制

  • 檔案處理
    • 選擇處理語言
    • 設定輸出格式
    • 啟動分析流程
  • 即時監控
    • 查看處理進度
    • 預覽處理結果
    • 下載完成檔案

9.3 實用場景

圖9-2：實用場景展示

會議記錄

  • 用手機錄音，自動轉為逐字稿
  • 拍攝簡報，即時生成摘要
  • 快速整理會議重點

文件處理

  • 拍攝紙本文件自動數位化
  • 批次處理多頁文件
  • 即時生成文件摘要

學習輔助

  • 課堂錄音自動轉錄
  • 拍攝黑板內容生成筆記
  • 快速製作學習重點

9.4 使用建議

網路設定

  • 確保手機和電腦在同一網路
  • 檢查防火牆設定
  • 優先使用 HTTPS 連線

效能優化

  • 大檔案建議使用 WiFi 上傳
  • 避免同時上傳過多檔案
  • 定期清理暫存檔案

安全建議

  • 僅在可信任的網路使用
  • 完成使用後關閉連線
  • 定期更新程式版本

9.5 常見問題解決

圖9-3：常見問題解決指南

連線問題

  • 無法掃描 QR Code
    • 確認相機權限
    • 檢查 QR Code 清晰度
    • 嘗試手動輸入網址
  • 連線失敗
    • 確認網路連接
    • 檢查防火牆設定
    • 確認端口是否開放

上傳問題

  • 檔案上傳失敗
    • 檢查檔案大小
    • 確認網路穩定性
    • 重試上傳程序
  • 處理速度慢
    • 檢查網路速度
    • 確認系統資源
    • 考慮分批處理
