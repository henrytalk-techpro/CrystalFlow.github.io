3.2 Model Selection Recommendations

General Text Processing

  • Recommended: gemini-2.0-flash-lite-001
    • Fast processing speed
    • Low resource consumption
    • Suitable for: Daily document processing, simple translation

Professional Content Analysis

  • Recommended: gpt-4o-mini or deepseek-r1
    • Strong understanding capability
    • More professional responses
    • Suitable for: Academic research, professional reports

Code Processing

  • Recommended: deepseek-coder-v2-lite-instruct
    • Specialized optimization for code understanding and generation
    • Suitable for: Code analysis, technical documentation

Mixed Image-Text Processing

  • Recommended: llava-llama3 or llava-v1.5-7b
    • Can process images and text simultaneously
    • Suitable for: Image-text reports, multimedia content analysis

3.3 Model Usage Considerations

System Resource Considerations

  • Local models require more memory
  • Cloud models require stable internet connection
  • Recommend choosing appropriate models based on computer configuration

Usage Costs

  • OpenRouter models may require additional fees
  • Local models are one-time download
  • Recommend testing with free versions first

Performance Optimization

  • Can switch between different models to compare effects
  • Supports hot-swapping of models
  • Recommend keeping model versions updated

3.4 Bilingual Mind Map Settings

Core Features

  • Bilingual Side-by-Side Display:
    • Perfect Chinese-English comparison
    • Automatic layout adjustment
    • Supports other language combinations
  • Customization Options:
    • Adjustable mind map levels (default 3 levels)
    • Custom node expansion methods
    • Flexible color theme settings
  • Multiple Output Formats:
    • HTML interactive viewing
    • PNG/SVG image export
    • PDF document output

Usage Recommendations

  • Academic Research: Set 4-5 levels depth for complete research structure presentation
  • Course Notes: Use 3-level structure to highlight key concepts
  • Meeting Records: 2-3 levels most suitable for quick grasp of main points
