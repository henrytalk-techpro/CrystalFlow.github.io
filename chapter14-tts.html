14.1 TTS 功能概述

圖14-1：TTS功能概覽

核心功能

  • 多提供商支援：
    • OpenAI TTS：高品質語音合成
    • Azure Speech Services：多語言支援
    • 智能切換和配置管理
  • 豐富的語音選項：
    • 多種語音風格和音色
    • 語速和音調調整
    • 多語言語音支援
  • 實用功能：
    • 即時語音生成
    • 音頻播放和保存
    • 批量處理功能

14.2 支援的 TTS 提供商

OpenAI TTS

  • 語音選項：
    • alloy：中性語音
    • echo：溫暖語音
    • fable：敘事語音
    • onyx：深沉語音
    • nova：清晰語音
    • shimmer：柔和語音
  • 模型選項：
    • tts-1：標準品質
    • tts-1-hd：高品質
  • 語速範圍：0.25 - 4.0

Azure Speech Services

  • 多語言支援：
    • 英文：Jenny, Guy
    • 中文：曉曉, 雲希
    • 日文：七海
    • 韓文：善熙
  • 語音類型：
    • 神經語音 (Neural)：高品質
    • 標準語音 (Standard)：穩定
  • 音調控制：x-low 到 x-high

14.3 使用方式

基本操作步驟

  1. 點擊 TTS 按鈕
    • 確保已設定 OpenAI API Key
    • 系統會自動檢查配置
  2. 選擇 TTS 提供商
    • OpenAI TTS：需要 OpenAI API Key
    • Azure Speech Services：需要 Azure 配置
  3. 設定語音參數
    • 選擇語音類型
    • 調整語速和音調
    • 選擇輸出格式
  4. 輸入文字並生成
    • 在文字框中輸入要轉換的文字
    • 點擊「生成語音」按鈕
    • 等待處理完成
  5. 播放和保存
    • 使用播放按鈕預覽
    • 保存音頻文件到指定位置

14.4 進階功能

批量處理

  • 支援多個文字同時處理
  • 自動生成序號文件名
  • 進度顯示和狀態更新
  • 批量保存到指定目錄

音頻播放控制

  • 即時播放預覽
  • 播放/暫停控制
  • 音量調節
  • 播放進度顯示

14.7 常見問題解決

圖14-2：TTS問題解決指南

API 相關問題

  • API Key 錯誤：
    • 檢查 API Key 是否正確
    • 確認帳戶額度是否充足
    • 檢查網路連接狀態
  • 生成失敗：
    • 檢查輸入文字是否為空
    • 確認文字長度限制
    • 重試生成操作

音頻播放問題

  • 無法播放：
    • 檢查音頻設備連接
    • 確認系統音量設定
    • 檢查音頻文件完整性
  • 音質問題：
    • 調整語音參數設定
    • 選擇高品質模型
    • 檢查輸出格式設定

14.8 最佳實踐建議

效能優化

  • 文字長度控制：
    • 建議單次處理不超過 5000 字元
    • 長文字建議分段處理
    • 使用批量處理功能
  • 網路優化：
    • 確保穩定的網路連接
    • 避免同時進行多個 TTS 請求
    • 合理使用 API 額度

品質提升

  • 語音選擇：
    • 根據內容類型選擇合適語音
    • 測試不同語音效果
    • 考慮目標聽眾偏好
  • 參數調整：
    • 適當調整語速和音調
    • 選擇高品質模型
    • 測試不同設定組合
