4.1 可選模型等級

圖4-1：Whisper模型等級選擇介面

small（建議選擇）

圖4-2：Small模型設定和效果展示

適用配置：

  • CPU：4核心以上
  • 記憶體：8GB RAM
  • 硬碟空間：約 1GB

特點：

  • 平衡效能與準確度
  • 適合一般用途
  • 處理速度適中

其他可選模型

tiny

最低配置需求：
  • CPU：雙核心
  • 記憶體：4GB RAM
  • 硬碟空間：約 500MB
適用場景：
  • 資源受限的系統
  • 簡單的語音辨識任務
  • 需要快速處理的場合

medium

建議配置：
  • CPU：6核心以上
  • 記憶體：16GB RAM
  • 硬碟空間：約 2.5GB
適用場景：
  • 需要較高準確度
  • 複雜的語音內容
  • 多語言處理

large

最低配置要求：
  • CPU：8核心以上
  • 記憶體：32GB RAM
  • 硬碟空間：約 5GB
  • 建議：NVIDIA GPU 6GB+ VRAM
適用場景：
  • 專業語音轉錄
  • 需要最高準確度
  • 複雜環境下的語音辨識

4.2 Whisper API 選項

圖4-3：Whisper API設定介面

使用時機

  • 電腦配備不足以運行本地模型
  • 需要更快的處理速度
  • 不想下載和儲存本地模型
  • 需要更穩定的辨識品質

價格說明（OpenAI Whisper API）

  • 一般音訊檔案：$0.006 / 分鐘
  • 例：1小時音訊約需 $0.36 美元

使用方式

  • 在介面上勾選 "Whisper API" 選項
  • 確保已設定有效的 OpenAI API Key
  • 直接上傳音訊檔案進行處理

優點

  • 不受本地硬體限制
  • 穩定的辨識品質
  • 支援多種音訊格式
  • 處理速度快

注意事項

  • 需要穩定的網路連接
  • 需要有足夠的 API 額度
  • 音訊檔案大小限制：25 MB
  • 建議先估算成本再使用

4.3 本地模型 vs API 選擇建議

考量面向 本地 Whisper 模型 Whisper API
成本 一次性下載，無額外費用 按使用量計費
硬體需求 較高 最低配備即可
處理速度 依硬體性能而定 快速且穩定
網路需求 下載後不需要 需要穩定網路
適用場景 大量本地處理、隱私需求高 臨時使用、硬體受限
