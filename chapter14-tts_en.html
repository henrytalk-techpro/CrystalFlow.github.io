14.3 Usage Methods

Basic Operation Steps

  1. Click TTS Button
    • Ensure OpenAI API Key is configured
    • System will automatically check configuration
  2. Select TTS Provider
    • OpenAI TTS: Requires OpenAI API Key
    • Azure Speech Services: Requires Azure configuration
  3. Configure Voice Parameters
    • Select voice type
    • Adjust speed and pitch
    • Choose output format
  4. Input Text and Generate
    • Enter text to convert in the text box
    • Click "Generate Speech" button
    • Wait for processing completion
  5. Play and Save
    • Use play button to preview
    • Save audio file to specified location

14.7 Common Issue Resolution

Figure 14-2: TTS Issue Resolution Guide

API Related Issues

  • API Key Error:
    • Check if API Key is correct
    • Confirm account has sufficient credits
    • Check network connection status
  • Generation Failed:
    • Check if input text is empty
    • Confirm text length limits
    • Retry generation operation

Audio Playback Issues

  • Unable to Play:
    • Check audio device connection
    • Confirm system volume settings
    • Check audio file integrity
  • Audio Quality Issues:
    • Adjust voice parameter settings
    • Select high-quality model
    • Check output format settings

14.8 Best Practice Recommendations

Performance Optimization

  • Text Length Control:
    • Recommend processing no more than 5000 characters at once
    • Long texts should be processed in segments
    • Use batch processing functionality
  • Network Optimization:
    • Ensure stable network connection
    • Avoid simultaneous multiple TTS requests
    • Use API credits reasonably

Quality Enhancement

  • Voice Selection:
    • Choose appropriate voice based on content type
    • Test different voice effects
    • Consider target audience preferences
  • Parameter Adjustment:
    • Appropriately adjust speed and pitch
    • Select high-quality models
    • Test different setting combinations
