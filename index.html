Crystal Flow AI Media Analyzer 使用手冊

1. 工具概述

Crystal Flow AI Media Analyzer 是一款專為分析多媒體資料而設計的智能工具。它支持處理多種格式的數位內容分析整理，包括 YouTube 影片、音頻檔案、文檔（TXT、PDF、PPT、EXL、Word 等），並通過視覺化心智圖、逐字稿和雙語翻譯等功能，幫助用戶高效率的分析並管理資訊。

2. 系統需求

  • 操作系統：Windows 10 或更高版本
  • 硬體要求：
    • 最低：雙核 CPU，4GB RAM
    • 推薦：四核 CPU，8GB RAM，具備 NVIDIA GPU（支持 CUDA）
  • 必要軟件：
    • Node.js：請參見 Node.js 安裝指南
    • Markdown file Reader：如 VsCode 或瀏覽器插件（Chrome、Microsoft Edge）
    • ffmpeg：開源程式外掛, 程式會自動下載安裝

3. 功能介紹

a. 雙使用者介面支援

Windows 使用者操作介面

  • 提供完整的多媒體分析功能，專為處理 PC 上的多媒體資訊設計。
  • 支持處理包括音頻、視頻、文檔和心智圖生成等功能。

網頁操作介面

  • 支援通過手機掃描 QR Code 快速上傳資料至 PC 進行分析。
  • 用戶可直接在手機端查看進度，並與 PC 工具無縫協作。

b. 網路多媒體知識解析

  • 支持多種來源與格式
    • YouTube：自動處理公開或私人連結的影片內容，提取字幕並進行分析。
      • 適用場景：學術課程筆記、技術演示摘要。
    • Facebook 影片：支持處理分享連結，將影片內容轉為可視化數據。
      • 適用場景：會議記錄、品牌廣告分析。
    • 任意 URL：支持解析數位文章（新聞、電子報、維基百科）。
      • 適用場景：研究核心內容提取。
  • 視訊字幕下載與翻譯
    • 自動提取 YouTube 和 Facebook 的字幕文件。
    • 提供多語言翻譯功能，匯出為 TXT 和 Markdown (MD) 文件。
    • 可生成 HTML 格式心智圖，供瀏覽器查看。

c. 圖文資料處理

  • 文本處理
    • 支持從 PDF、PPT、Word 和 Excel 文件中提取內容，轉換為 Markdown 格式。
      • 適用場景：學術研究資料整理、商業報告準備。
    • 準確保留表格、段落、標題等原始排版。
    • 用戶可直接生成心智圖或進行多語言翻譯。
  • 圖片處理
    • 支援 jpg、jpeg、png、bmp、heic、webp 格式。
    • 從圖片或掃描文件中提取文字，轉為 Markdown 格式。
      • 適用場景：手寫筆記數位化、圖片文字分析。
    • 結合 Google Lens/Microsoft Lens，支持快速掃描與文字化。

d. 本地音頻與 MP3 音樂處理

  • 支持 MP3、M4A、ACC 和 WAV 格式。
  • 提取歌詞並翻譯為雙語對照。
    • 適用場景：歌曲翻譯學習、音樂資料整理。
  • 利用語音轉文字技術生成 Markdown 格式逐字稿。
  • 逐字稿或歌詞結構可生成心智圖。

e. PC 錄音功能

  • 即時錄音處理多媒體資料，支持錄音自動轉逐字稿。
    • 適用場景：會議錄音分析、演講逐字稿生成。
  • 錄音時長自定義，背景錄音與時間戳標記。

f. 心智圖生成

  • 自定義層級深度與結構化呈現。
    • 適用場景：學術研究、會議記錄、創意思考。
  • 結合 GPT 分析與多語言支持。

g. 雙語翻譯與朗讀

  • 支持多語言翻譯與雙語對照輸出。
    • 適用場景：國際會議資料準備、多語學習輔助。
  • 提供朗讀功能，調整語速、語調與語音風格。
  • 匯出為 Markdown、TXT 或 PDF 文件。

5. 操作指南

初始化設置

  • 啟動軟件後，輸入有效的 OpenAI API Key。
  • 確保 NVIDIA 驅動已安裝（若啟用 GPU 加速）。
  • 在 Google Sheets 中註冊 email，啟用試用期。

核心功能使用

  • 分析 YouTube 或 URL: 粘貼影片連結至 "輸入媒體網址" 欄位。
  • 選擇下載字幕語言。
  • 點擊 "分析文本" 按鈕。

心智圖生成

  • 選擇心智圖層數。
  • 點擊 "分析文本"，生成的心智圖自動保存。

錄音與逐字稿生成

  • 點擊錄音按鈕，開始錄音。
  • 錄音完成後，自動生成逐字稿。

6. 進階功能

  • 支持多語言解析與翻譯。
  • 動態分段處理與 token 數調整。
  • 與 GPT-4o-mini 的互動。

8. 常見問題解答 (FAQ)

  1. 按鈕無法點擊？

  2. 心智圖太簡略？
