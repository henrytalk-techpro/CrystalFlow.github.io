8.1 Process Network Info. 功能說明

圖8-1：網路媒體處理介面總覽

支援的網路媒體

YouTube 影片

  • 自動檢測是否有字幕
  • 有字幕：直接下載並處理字幕
  • 無字幕：使用 Whisper 模型轉錄語音

Facebook 內容

  • 支援一般影片
  • 支援 Facebook Reels
  • 自動使用 Whisper 模型處理音訊

8.2 處理流程

圖8-2：網路媒體處理流程

步驟 1：輸入媒體網址

  • 在 "Enter Media URL" 欄位貼上連結
  • 支援 YouTube、Facebook 等平台的連結
  • 自動識別連結類型

步驟 2：選擇處理選項

  • Download Subtitles：下載現有字幕
  • Transcription Mode：語音轉文字模式
  • 選擇目標語言

步驟 3：開始處理

  • 點擊 "Process Network Info." 開始處理
  • 自動判斷處理方式
  • 顯示處理進度
  • 完成後自動輸出結果

8.3 使用注意事項

網址格式

  • 確保輸入完整的影片網址
  • 支援標準分享連結格式
  • 避免使用短網址

處理時間

  • 依據影片長度而定
  • 使用 Whisper 模型時會較久
  • 建議預留足夠處理時間

網路需求

  • 需要穩定的網路連接
  • 建議使用高速網路
  • 避免網路中斷

8.4 常見問題處理

圖8-3：常見問題處理指南

網址無效

  • 檢查網址是否完整
  • 確認影片是否可公開訪問
  • 檢查地區限制

下載失敗

  • 檢查網路連接
  • 確認系統資源充足
  • 重試下載程序

處理中斷

  • 檢查網路穩定性
  • 確認系統未進入休眠
  • 重新開始處理

8.5 最佳實踐建議

效能優化

  • 避免同時處理多個長視頻
  • 定期清理暫存檔案
  • 監控系統資源使用

結果管理

  • 建立合適的檔案命名規則
  • 定期備份重要內容
  • 整理輸出檔案
