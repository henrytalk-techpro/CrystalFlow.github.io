系統需求

  • 操作系統：Windows
  • 硬體要求：至少 1GB RAM
  • 麥克風：支援語音輸入功能
  • 網路連接：需要下載安裝包

安裝步驟

1. 下載 Node.js

訪問 Node.js 官方網站，選擇適合您操作系統的安裝包進行下載。

2. 安裝 Node.js

根據您的操作系統，按照以下步驟進行安裝：

  • Windows：雙擊下載的 .msi 文件，並按照安裝向導的指示進行安裝。

3. 安裝 markmap-cli

安裝完成 Node.js 後，您可以使用 npm 安裝 markmap-cli。請在終端或命令提示符中輸入以下命令：

npm install -g markmap-cli

這將全局安裝 markmap-cli，您可以在任何地方使用它。

4. 驗證安裝

安裝完成後，您可以在終端或命令提示符中輸入以下命令來驗證 markmap-cli 是否安裝成功：

markmap -v

如果顯示版本號，則表示安裝成功。

常見問題解答

  1. 如何卸載 Node.js？
    • Windows：通過控制面板的"程序和功能"卸載。
  2. Node.js 和 npm 有什麼區別？

    Node.js 是一個 JavaScript 執行環境，而 npm 是 Node.js 的包管理器，用於安裝和管理 Node.js 的庫和工具。
