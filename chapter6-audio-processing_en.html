6.1 Supported Audio Formats

Figure 6-1: Supported Audio Format Overview

MP3

  • Most common audio compression format
  • Supports various bit rates
  • Suitable for music and speech

WAV

  • Lossless audio format
  • High-quality audio preservation
  • Suitable for professional audio processing

M4A

  • Apple audio format
  • Better compression efficiency
  • Supports metadata tagging

AAC

  • Advanced Audio Coding
  • High-efficiency compression
  • Widely used in digital audio

6.5 Usage Recommendations

Figure 6-5: Usage Recommendations and Best Practices

Audio Quality Recommendations

  • Recommended sample rate: 16kHz or higher
  • Recommended bit rate: 128kbps or higher
  • Avoid excessive background noise

Processing Tips

  • For long audio, recommend segment processing
  • Choose appropriate transcription mode
  • Use suitable Whisper model

Performance Optimization

  • For large files, recommend using GPU acceleration
  • Monitor memory usage during batch processing
  • Regularly clean temporary files
