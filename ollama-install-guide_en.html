Ollama Installation Guide

1. System Requirements

  • Operating System: Windows 10 or higher, macOS, or Linux
  • At least 4GB of RAM
  • Internet connection required

2. Installation Steps

Windows

  1. Go to the Ollama official website to download the latest Windows installer.
  2. Double-click the downloaded installer and follow the on-screen instructions to complete the installation.

macOS

  1. Open the terminal and execute the following command: 
    brew install ollama/tap/ollama
  2. Run the following command to confirm the installation was successful: 
    ollama --version

Linux

  1. Install Ollama using the appropriate package manager for your distribution.
  2. Run the following command to confirm the installation was successful: 
    ollama --version

3. Basic Usage

After installation, you can use the following command to start Ollama:

ollama start

4. Installation Experience

Here is my experience while installing Ollama:

@echo off

REM First, stop the Ollama service, forcefully terminate the ollama.exe process, and wait 2 seconds to ensure the service has completely stopped
taskkill /F /IM ollama.exe 2>nul
timeout /t 5 /nobreak >nul

REM First, close the command line output to avoid displaying command output information during execution
set "CUDA_PATH=C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v11.8"
set "CUDA_HOME=%CUDA_PATH%"

REM Next, set up the CUDA runtime by adding the binary and library paths of CUDA to the system PATH variable
set "PATH=%CUDA_PATH%\bin;%CUDA_PATH%\lib\x64;%PATH%"

REM Then, configure Ollama's CUDA settings, including setting visible CUDA devices, enabling CUDA, specifying device type, setting GPU layers, and GPU overhead
set "CUDA_VISIBLE_DEVICES=0"
REM Enable CUDA acceleration
set "OLLAMA_CUDA=1" 
REM Specify using CUDA devices for computation
set "OLLAMA_DEVICE=cuda"
REM Set GPU layers, affecting the model's computational efficiency and accuracy
set "OLLAMA_GPU_LAYERS=35"
REM Set GPU overhead, affecting the model's computational efficiency and accuracy 
set "OLLAMA_GPU_OVERHEAD=1" 
set "OLLAMA_HOST=http://127.0.0.1:11434"

REM Additionally, set the CUDA version used by Ollama
set "OLLAMA_CUDA=11.8"

REM Finally, start the Ollama service
ollama serve
