OpenAI API Key 申請指南

  • 申請條件
  • 申請步驟
  • 常見問題解答

申請條件

  • 擁有有效的 OpenAI 帳戶。
  • 同意 OpenAI 的使用條款和隱私政策。

申請步驟

1. 登入 OpenAI 帳戶

訪問 OpenAI 登入頁面，輸入您的帳戶資訊進行登入。

2. 進入 API Keys 設定頁面

登入後，前往 API Keys 設定頁面。

3. 創建新的 API Key

在 API Keys 頁面，點擊 + Create new secret key 按鈕，系統將自動生成一個新的 API Key。

4. 保存您的 API Key

生成的 API Key 將顯示在列表中，請務必將其妥善保存。切勿與他人分享此密鑰。

常見問題解答

  1. 我可以擁有多個 API Key 嗎？

    是的，您可以根據需要創建多個 API Key。

  2. 如果我丟失了 API Key，該怎麼辦？

    您可以隨時在 API Keys 頁面生成新的密鑰，並刪除舊的密鑰。

  3. API Key 有效期是多久？

    API Key 沒有固定的有效期，但如果發現密鑰被公開，OpenAI 可能會禁用該密鑰。

  4. GPT-4o-mini API 如何計費？

    GPT-4o-mini API 的計費方式通常是根據使用的 token 數量來計算的。具體的價格如下：

    • 輸入 tokens：$0.150 / 1M tokens
    • 緩存的輸入 tokens：$0.075 / 1M tokens
    • 輸出 tokens：$0.600 / 1M tokens

    使用 Batch API 時的價格如下：

    • 輸入 tokens：$0.075 / 1M tokens
    • 輸出 tokens：$0.300 / 1M tokens

    具體的價格和計費細則可以參考 OpenAI 的官方定價頁面。
