申請要求

  • 擁有有效的 OpenAI 帳號
  • 同意 OpenAI 的服務條款和隱私政策

申請步驟

1. 登入您的 OpenAI 帳號

前往 OpenAI 登入頁面 並輸入您的帳號資訊進行登入。

2. 前往 API Keys 設定頁面

登入後，前往 API Keys 設定頁面。

3. 建立新的 API Key

在 API Keys 頁面上，點擊 + Create new secret key 按鈕，系統將自動生成新的 API Key。

4. 儲存您的 API Key

生成的 API Key 將顯示在列表中，請務必妥善保存。請勿與他人分享此金鑰。

常見問題

  1. 我可以擁有多個 API Keys 嗎？

    是的，您可以根據需要建立多個 API Keys。

  2. 如果我遺失了 API Key 該怎麼辦？

    您可以隨時在 API Keys 頁面生成新的金鑰，並刪除舊的金鑰。

  3. API Key 的有效期限是多久？

    API Key 沒有固定的有效期限，但如果發現金鑰外洩，OpenAI 可能會停用它。

  4. GPT-4o-mini API 如何計費？

    GPT-4o-mini API 的計費通常是根據使用的 token 數量計算。具體價格如下：

    • 輸入 tokens：$0.150 / 1M tokens
    • 快取輸入 tokens：$0.075 / 1M tokens
    • 輸出 tokens：$0.600 / 1M tokens

    使用 Batch API 的價格如下：

    • 輸入 tokens：$0.075 / 1M tokens
    • 輸出 tokens：$0.300 / 1M tokens

    具體價格和計費細節，請參考 OpenAI 的官方定價頁面。
