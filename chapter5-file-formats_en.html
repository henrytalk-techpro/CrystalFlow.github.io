5.1 Image Format Support

Figure 5-1: Supported Image Format Examples

Standard Image Formats

  • JPG/JPEG
    • Most common image format
    • Suitable for photos and complex images
    • Supports various resolutions
  • PNG
    • Supports transparent background
    • Suitable for screenshots and text images
    • Lossless compression format
  • BMP
    • Uncompressed format
    • Preserves original image quality completely

Special Image Formats

  • HEIC/HEIF
    • Common format for iOS devices
    • Automatically converts to JPG for processing
    • Maintains original image quality
  • WebP
    • Modern image format developed by Google
    • Supports animated images
    • Higher compression efficiency

5.2 Document Format Support

Figure 5-2: Supported Document Format Overview

Document Processing Files

  • PDF
    • Supports multi-page documents
    • Can select page range for processing
    • Supports mixed text and image content
    • Can process scanned documents
  • Microsoft Office Formats
    • Word (.doc, .docx)
      • Complete text format support
      • Preserves paragraph structure
      • Supports table conversion
    • PowerPoint (.ppt, .pptx)
      • Supports slide content extraction
      • Preserves presentation structure
      • Can process embedded images
    • Excel (.xls, .xlsx)
      • Table data processing
      • Multi-worksheet support
      • Data format preservation

Plain Text Files

  • TXT
    • Supports multiple encodings (UTF-8, Big5, etc.)
    • Automatic encoding detection
    • Suitable for large text processing
  • Markdown (.md)
    • Preserves format markers
    • Supports conversion to other formats
    • Suitable for technical documentation

5.3 Processing Recommendations

Figure 5-3: File Processing Recommendations

Image Processing Recommendations

  • Image size: Recommend not exceeding 10MB
  • Resolution: Recommend 300-1200 DPI
  • Text images: Ensure text is clear and readable

Document Processing Recommendations

  • PDF Documents:
    • Recommend confirming if it's a text-based PDF first
    • For scanned documents, ensure scanning quality
    • Can selectively process page ranges
  • Office Documents:
    • Recommend using newer format versions
    • Ensure documents are not encrypted
    • Complex formats may require preprocessing
