Crystal Flow AI Media Analyzer User Manual

1. Overview

Crystal Flow AI Media Analyzer is an intelligent tool designed for analyzing multimedia data. It supports processing various formats of digital content, including YouTube videos, audio files, documents (TXT, PDF, PPT, EXL, Word, etc.), and helps users efficiently analyze and manage information through mind maps, transcripts, and bilingual translations.

  • Operating System: Windows 10 or higher
  • Hardware Requirements:
    • Minimum: Dual-core CPU, 4GB RAM
    • Recommended: Quad-core CPU, 8GB RAM, NVIDIA GPU (CUDA support, optional)
  • Required Software:

3. Features

a. Dual User Interface Support

  • Windows User Interface
    • Provides complete multimedia analysis functionality, designed for processing multimedia information on PC.
    • Supports processing audio, video, documents, and mind map generation.
  • Web Interface
    • Supports quick data upload to PC via QR Code scanning from mobile devices.
    • Users can check progress directly on mobile and seamlessly collaborate with PC tools.

b. Online Multimedia Knowledge Analysis

  • Supports Multiple Sources and Formats
    • YouTube: Automatically processes public or private video content, extracts subtitles for analysis.
      • Use Cases: Academic course notes, technical demonstration summaries.
    • Facebook Videos: Processes shared links, converts video content into visualized data.
      • Use Cases: Meeting records, brand advertising analysis.
    • Any URL: Supports parsing digital articles (news, newsletters, Wikipedia).
      • Use Cases: Research core content extraction.
  • Video Subtitle Download and Translation
    • Automatically extracts subtitles from YouTube and Facebook.
    • Provides multi-language translation, exports to TXT and Markdown (MD) files.
    • Can generate HTML format mind maps for browser viewing.

c. Text and Image Processing

  • Text Processing
    • Supports content extraction from PDF, PPT, Word, and Excel files, converts to Markdown format.
      • Use Cases: Academic research organization, business report preparation.
    • Accurately preserves tables, paragraphs, and heading layouts.
    • Users can directly generate mind maps or perform multi-language translations.
  • Image Processing
    • Supports jpg, jpeg, png, bmp, heic, webp formats.
    • Extracts text from images or scanned documents, converts to Markdown format.
      • Use Cases: Handwritten note digitization, image text analysis.
    • Integrates with Google Lens/Microsoft Lens for quick scanning and text conversion.

d. Local Audio and MP3 Music Processing

  • Supports MP3, M4A, ACC, and WAV formats.
  • Extracts lyrics and provides bilingual translations.
    • Use Cases: Song translation learning, music data organization.
  • Generates Markdown format transcripts using speech-to-text technology.
  • Can create mind maps from transcripts or lyrics structure.

e. PC Recording Function

  • Real-time audio recording and processing, supports automatic transcript generation.
    • Use Cases: Meeting recording analysis, speech transcript generation.
  • Customizable recording duration, background recording, and timestamp marking.

f. Mind Map Generation

  • Customizable depth levels and structured presentation.
    • Use Cases: Academic research, meeting records, creative thinking.
  • Combines GPT analysis with multi-language support.

g. Bilingual Translation and Text-to-Speech

  • Supports multi-language translation and bilingual parallel output.
    • Use Cases: International conference material preparation, multilingual learning assistance.
  • Provides text-to-speech functionality with adjustable speed, tone, and voice style.
  • Exports to Markdown, TXT, or PDF files.

5. User Guide

Initial Setup

  • Enter a valid OpenAI API Key after launching the software.
  • Ensure NVIDIA drivers are installed (if GPU acceleration is enabled).
  • Register email in Google Sheets to activate trial period.

Core Feature Usage

  • Analyze YouTube or URL: Paste video link into the "Enter Media URL" field.
  • Select subtitle language for download.
  • Click "Analyze Text" button.

Mind Map Generation

  • Choose mind map depth levels.
  • Click "Analyze Text", generated mind map saves automatically.

Recording and Transcript Generation

  • Click the record button to start recording.
  • Transcript generates automatically after recording completion.
