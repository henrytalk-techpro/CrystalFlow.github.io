Ollama 安裝指南

1. 系統需求

  • 作業系統：Windows 10 或更高版本、macOS 或 Linux
  • 至少 4GB 記憶體
  • 需要網路連線

2. 安裝步驟

Windows

  1. 前往 Ollama 官方網站 下載最新的 Windows 安裝程式。
  2. 雙擊下載的安裝程式，並依照螢幕上的指示完成安裝。

macOS

  1. 開啟終端機並執行以下指令： 
    brew install ollama/tap/ollama
  2. 執行以下指令確認安裝是否成功： 
    ollama --version

Linux

  1. 使用適合您發行版本的套件管理器安裝 Ollama。
  2. 執行以下指令確認安裝是否成功： 
    ollama --version

3. 基本使用

安裝完成後，您可以使用以下指令啟動 Ollama：

ollama start

4. 安裝經驗

以下是我安裝 Ollama 的經驗：

@echo off

REM 首先，停止 Ollama 服務，強制終止 ollama.exe 進程，並等待 2 秒確保服務完全停止
taskkill /F /IM ollama.exe 2>nul
timeout /t 5 /nobreak >nul

REM 首先，關閉命令列輸出，避免執行時顯示命令輸出資訊
set "CUDA_PATH=C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v11.8"
set "CUDA_HOME=%CUDA_PATH%"

REM 接著，設定 CUDA 運行環境，將 CUDA 的二進制和函式庫路徑加入系統 PATH 變數
set "PATH=%CUDA_PATH%\bin;%CUDA_PATH%\lib\x64;%PATH%"

REM 然後，配置 Ollama 的 CUDA 設定，包括設定可見 CUDA 裝置、啟用 CUDA、指定裝置類型、設定 GPU 層數和 GPU 開銷
set "CUDA_VISIBLE_DEVICES=0"
REM 啟用 CUDA 加速
set "OLLAMA_CUDA=1" 
REM 指定使用 CUDA 裝置進行運算
set "OLLAMA_DEVICE=cuda"
REM 設定 GPU 層數，影響模型運算效率和精確度
set "OLLAMA_GPU_LAYERS=35"
REM 設定 GPU 開銷，影響模型運算效率和精確度
set "OLLAMA_GPU_OVERHEAD=1" 
set "OLLAMA_HOST=http://127.0.0.1:11434"

REM 另外，設定 Ollama 使用的 CUDA 版本
set "OLLAMA_CUDA=11.8"

REM 最後，啟動 Ollama 服務
ollama serve

ollama help
