Ollama 安裝指南

1. 系統需求

  • 作業系統：Windows 10 或更高版本，macOS，或 Linux
  • 至少 4GB 的 RAM
  • 具備網際網路連接

2. 安裝步驟

Windows

  1. 前往 Ollama 官方網站 下載最新的 Windows 安裝程式。
  2. 雙擊下載的安裝程式，並按照螢幕上的指示完成安裝。

macOS

  1. 打開終端機，執行以下命令： 
    brew install ollama/tap/ollama
  2. 執行以下命令確認安裝成功： 
    ollama --version

Linux

  1. 根據您的發行版，使用相應的包管理器安裝 Ollama。
  2. 執行以下命令確認安裝成功： 
    ollama --version

3. 基本使用

安裝完成後，您可以使用以下命令來啟動 Ollama：

ollama start

4. 安裝經驗

以下是我在安裝 Ollama 時的經驗：

@echo off

REM 之後，停止 Ollama 服務，強制終止 ollama.exe 進程，並等待 2 秒以確保服務已完全停止
taskkill /F /IM ollama.exe 2>nul
timeout /t 5 /nobreak >nul

REM 首先，關閉命令行輸出，以免在執行過程中顯示命令的輸出信息
set "CUDA_PATH=C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v11.8"
set "CUDA_HOME=%CUDA_PATH%"

REM 接著，進行 CUDA 運行時的設置，將 CUDA 的二進制文件路徑和庫路徑添加到系統的 PATH 變量中
set "PATH=%CUDA_PATH%\bin;%CUDA_PATH%\lib\x64;%PATH%"

REM 然後，進行 Ollama 的 CUDA 設置，包括設定可見的 CUDA 設備、啟用 CUDA、指定設備類型、設定 GPU 層數和 GPU 開銷
set "CUDA_VISIBLE_DEVICES=0"
REM 啟用 CUDA 加速
set "OLLAMA_CUDA=1" 
REM 指定使用 CUDA 設備進行運算
set "OLLAMA_DEVICE=cuda"
REM 設置 GPU 層數，影響模型的計算效率和精度
set "OLLAMA_GPU_LAYERS=35"
REM 設置 GPU 開銷，影響模型的計算效率和精度 
set "OLLAMA_GPU_OVERHEAD=1" 
set "OLLAMA_HOST=http://127.0.0.1:11435"

REM 再者，設置 Ollama 使用的 CUDA 版本
set "OLLAMA_CUDA=11.8"

REM 最後，啟動 Ollama 服務
ollama serve
