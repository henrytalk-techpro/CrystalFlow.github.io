OpenRouter API Key 申請指南

1. 什麼是 OpenRouter？

OpenRouter 是一個統一的 API 平台，讓您可以輕鬆存取多個 AI 語言模型，包括 Anthropic Claude、GPT-4、Google PaLM 等。使用 OpenRouter 可以：

  • 統一存取多個 AI 模型
  • 更優惠的價格
  • 簡化的 API 整合

2. 申請步驟

  1. 訪問 OpenRouter 官網
    • 前往 https://openrouter.ai/
  2. 註冊帳號
    • 點擊右上角的 "Sign Up"
    • 使用 GitHub 或 Google 帳號登入
  3. 獲取 API Key
    • 登入後進入 Dashboard
    • 在 "API Keys" 部分找到您的 API Key
    • 如果沒有現有的 Key，點擊 "Create New Key" 創建

3. 費用說明

  • 新用戶可獲得 $5 免費額度
  • 採用預付費模式，可根據需求充值
  • 不同模型有不同的價格，詳情請參考 OpenRouter 官網價格頁面

4. 安全建議

  • 請妥善保管您的 API Key，不要分享給他人
  • 定期檢查 API 使用情況
  • 如發現異常使用，及時在 Dashboard 中重置 API Key

5. 常見問題

Q: API Key 無法使用？
A: 請確認：
  • 帳戶餘額是否充足
  • API Key 是否正確複製
  • API Key 是否仍然有效
Q: 如何查看使用記錄？
A: 在 OpenRouter Dashboard 中可以查看詳細的使用記錄和費用明細。
Q: 支援哪些付款方式？
A: OpenRouter 支援信用卡和部分數位支付方式，具體以官網顯示為準。
