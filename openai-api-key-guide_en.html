Application Requirements

  • Have a valid OpenAI account.
  • Agree to OpenAI's terms of service and privacy policy.

Application Steps

1. Log in to your OpenAI account

Visit the OpenAI login page and enter your account information to log in.

2. Go to the API Keys settings page

After logging in, go to the API Keys settings page.

3. Create a new API Key

On the API Keys page, click the + Create new secret key button, and the system will automatically generate a new API Key.

4. Save your API Key

The generated API Key will be displayed in the list, please make sure to save it properly. Do not share this key with others.

Frequently Asked Questions

  1. Can I have multiple API Keys?

    Yes, you can create multiple API Keys as needed.

  2. What should I do if I lose my API Key?

    You can generate a new key at any time on the API Keys page and delete the old key.

  3. What is the validity period of the API Key?

    The API Key does not have a fixed validity period, but if the key is found to be exposed, OpenAI may disable it.

  4. How is the GPT-4o-mini API billed?

    The billing for the GPT-4o-mini API is typically calculated based on the number of tokens used. The specific prices are as follows:

    • Input tokens: $0.150 / 1M tokens
    • Cached input tokens: $0.075 / 1M tokens
    • Output tokens: $0.600 / 1M tokens

    The prices for using the Batch API are as follows:

    • Input tokens: $0.075 / 1M tokens
    • Output tokens: $0.300 / 1M tokens

    For specific prices and billing details, please refer to OpenAI's official pricing page.
