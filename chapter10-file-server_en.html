10.3 Network Setup Recommendations

Figure 10-2: Network Setup Guide

Basic Settings

  • Router settings:
    • Recommend using 5GHz WiFi
    • Place router in central location
    • Avoid signal interference
  • Connection methods:
    • Prioritize HTTPS connection
    • Support HTTP backup connection
    • Quick connection via QR Code

Security Recommendations

  • Network security:
    • Only use on home network
    • Set WiFi password protection
    • Regularly change network password
  • File security:
    • Regularly clean unnecessary files
    • Backup important documents promptly
    • Pay attention to privacy data protection

10.4 Performance Optimization Recommendations

File Management

  • Regular organization:

    • Maintain sufficient storage space
  • Usage recommendations:

Network Optimization

  • Transfer performance:

  • Multiple user usage:

10.5 Common Issue Resolution

Figure 10-3: Common Issue Resolution Guide

Connection Issues

  • Unable to access server:
    • Check network connection
    • Confirm devices are on same network
    • Check firewall settings
  • Slow transfer speed:
    • Check network bandwidth
    • Confirm file size
    • Consider using wired connection

File Issues

  • Files cannot be uploaded:
    • Check file size limits
    • Confirm sufficient storage space
    • Check file format support
  • Files cannot be previewed:
    • Confirm file format
    • Check browser support
    • Try downloading and viewing
