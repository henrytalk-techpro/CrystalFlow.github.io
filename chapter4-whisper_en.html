  • Suitable for general purposes
  • Moderate processing speed

Other Available Models

4.2 Whisper API Options

Figure 4-3: Whisper API Settings Interface

When to Use

  • Computer specifications insufficient to run local models
  • Need faster processing speed
  • Don't want to download and store local models
  • Need more stable recognition quality

Pricing Information (OpenAI Whisper API)

  • General audio files: $0.006 / minute
  • Example: 1 hour of audio costs approximately $0.36 USD

How to Use

  • Check the "Whisper API" option in the interface
  • Ensure a valid OpenAI API Key is configured
  • Directly upload audio files for processing

Advantages

  • Not limited by local hardware
  • Stable recognition quality
  • Supports multiple audio formats
  • Fast processing speed

Considerations

  • Requires stable internet connection
  • Need sufficient API quota
  • Audio file size limit: 25 MB
  • Recommend estimating costs before use

4.3 Local Model vs API Selection Recommendations

Consideration Local Whisper Model Whisper API
Cost One-time download, no additional fees Pay per usage
Hardware Requirements Higher Minimum specifications sufficient
Processing Speed Depends on hardware performance Fast and stable
Network Requirements Not needed after download Requires stable network
Suitable Scenarios Large-scale local processing, high privacy requirements Temporary use, hardware limitations
