11.1 RAG Feature Overview

Figure 11-1: RAG Feature Overview

Basic Principles

  • Knowledge Base Creation:
    • Use LM Studio's embedding model for vectorization
    • Automatically process and store document content
    • Build high-performance vector database
  • Intelligent Retrieval:
    • Automatically find the most relevant 5 pieces of information
    • Combine with AI model for analysis
    • Provide more accurate answers

11.3 Usage Methods

Knowledge Base Creation

  • Data Processing Workflow:
    • Upload or input URL
    • System automatically converts text
    • Vectorize and store
  • Automatic Updates:
    • New data automatically processed
    • Real-time knowledge base updates
    • Keep data current

Q&A Operations

  • General Conversation:
    • Enter questions directly in chat box
    • Press Enter to send (without using RAG)
    • Get general AI responses
  • RAG Enhanced Responses:
    • Enter question and click RAG button
    • System automatically retrieves relevant information
    • Combine knowledge base content for answers
