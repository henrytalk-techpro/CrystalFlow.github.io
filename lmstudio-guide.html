LM Studio 安裝與設定指南

1. LM Studio 簡介

LM Studio 是一個強大的本地 AI 模型管理工具，讓您能夠在本機上運行各種 AI 模型，無需依賴雲端服務。主要功能包括：

  • 支援多種 AI 模型
  • 提供本地 API 服務
  • 使用者友善介面
  • GPU 加速支援

2. 系統需求

  • 作業系統：Windows 10/11、macOS 或 Linux
  • 硬體需求：
    • 最低配備：8GB RAM、支援 AVX2 的 CPU
    • 建議配備：16GB+ RAM、NVIDIA GPU（支援 CUDA）
    • 儲存空間：至少 10GB 可用空間

3. 安裝步驟

  1. 下載 LM Studio
    • 前往 https://lmstudio.ai/
    • 選擇與您系統相容的版本
  2. 安裝過程
    • 執行下載的安裝程式
    • 依照安裝精靈的指示進行
    • 首次啟動可能需要管理員權限

4. 設定 text-embedding-nomic-embed-text-v1.5 模型

  1. 啟動 LM Studio
  2. 下載模型
    • 點擊左側的模型庫圖示
    • 搜尋 "text-embedding-nomic-embed-text-v1.5"
    • 點擊下載按鈕（約 84.11 MB）
  3. 設定本地伺服器
    • 在主介面選擇已下載的模型
    • 點擊「設定」按鈕檢查配置：
      • 上下文長度：2048
      • GPU 卸載：12/12（如果有 GPU）
      • 評估批次大小：512
    • 確保「狀態：運行中」顯示綠色
    • 伺服器地址將顯示為：http://127.0.0.1:11435

5. 驗證設定

  1. 檢查伺服器狀態
    • 確認狀態顯示「就緒」
    • 檢查開發者日誌是否有錯誤訊息
  2. 測試 API 連接
    • 使用提供的 cURL 指令進行測試
    • 驗證回應是否正常

6. 故障排除

問：模型無法載入？
答：檢查：
  • 系統記憶體是否足夠
  • GPU 驅動程式是否更新
  • 磁碟空間是否充足
問：API 連接失敗？
答：確認：
  • 伺服器狀態是否運行中
  • 端口 11435 是否被其他應用程式佔用
  • 防火牆設定是否允許連接

7. 進階設定

  • GPU 記憶體管理
    • 調整 GPU 卸載設定以優化效能
    • 監控 GPU 使用率
  • 效能優化
    • 調整批次大小以平衡速度和記憶體使用
    • 使用「保持模型在記憶體中」選項以加快回應時間
